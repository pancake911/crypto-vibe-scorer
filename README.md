# Crypto Vibe Scorer - 代币打分系统

一个专业的加密货币量化交易辅助决策仪表盘，帮助交易者根据多个数据维度对代币进行综合评分。

## 功能特性

### 自动数据获取
- **资金费率** (Binance Futures API)
- **多空持仓比** (Binance Futures)
- **恐惧贪婪指数** (Alternative.me API)
- **实时价格** (Binance)

### 手动观察输入
- **OI形态分析** (未平仓合约形态)
- **ETF资金流向** (手动输入)
- **盘口挂单特征** (多选)

### 智能评分系统
- 综合0-100分评分
- 详细得分报告
- 买入/持有/卖出建议
- Bloomberg Terminal风格界面

## 技术栈

- **Next.js 14** - React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **ccxt** - 加密货币交易所API库
- **Recharts** - 图表库（可选）

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

```bash
npm run dev
```

### 3. 打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

## 使用说明

1. **输入代币名称** (如 BTC, ETH)
2. **点击"获取数据"** 自动获取实时数据
3. **填写手动观察** (OI形态、ETF流向、盘口特征)
4. **点击"计算综合得分"** 查看评分和建议

## 评分规则

### 资金费率
- \> 0.10%: -20分 (极度危险)
- 0.07%-0.09%: -10分 (预警)
- 0%-0.02%: +10分 (健康)
- -0.01%到-0.04%: +20分 (最佳加仓)
- < -0.05%: +30分 (无脑抄底)

### 多空持仓比
- \> 2.5: -20分 (多头拥挤)
- < 0.4: +20分 (空头拥挤)

### 恐惧贪婪指数
- \> 90: -20分 (卖出)
- < 10: +30分 (买入)

### OI形态
- 价格新高 + OI新高: +10分
- 价格新高 + OI回落: -15分
- 价格大跌 + OI暴涨: +20分
- 价格横盘 + OI缓升: +10分

### ETF资金流向
- \> 10亿: +30分
- \> 5亿: +15分
- 连续流出 > 3亿: -15分

### 盘口挂单特征
- 真护盘: +10分
- 假护盘: -5分
- 真突破: +15分
- 假支撑: -15分

## 建议等级

- **> 75分**: 🟢 强烈买入 (Strong Buy)
- **40-75分**: 🟡 持有/观望 (Hold)
- **< 40分**: 🔴 减仓/卖出 (Sell)

# 🎓 完全小白教程 - 如何运行这个程序

## 📖 首先理解两个概念

### 1. "安装依赖"是什么意思？

**简单理解：**
- 这个程序需要很多"工具"才能运行
- 这些"工具"就是"依赖"
- "安装依赖"就是把所有需要的工具下载到你的电脑上
- 就像你买了一个玩具，需要先安装电池才能玩

**具体操作：**
- 打开"终端"（Terminal）
- 输入命令：`npm install`
- 等待下载完成（可能需要几分钟）

### 2. "运行开发服务器"是什么意思？

**简单理解：**
- 程序写好了，需要"启动"它才能使用
- "运行开发服务器"就是启动程序
- 启动后，你可以在浏览器里打开这个程序
- 就像你打开一个游戏软件一样

**具体操作：**
- 在终端输入命令：`npm run dev`
- 程序会启动
- 浏览器会自动打开，或者你手动访问 http://localhost:3000

## 🚀 完整操作步骤（超详细）

### 第一步：检查是否安装了Node.js

**什么是Node.js？**
- 这是运行这个程序必须的软件
- 就像运行Word需要安装Office一样

**如何检查？**

1. **打开"终端"（Terminal）**
   - Mac用户：按 `Command + 空格键`，输入"终端"，回车
   - 或者：应用程序 → 实用工具 → 终端

2. **输入命令检查**
   - 在终端里输入：`node --version`
   - 按回车
   - 如果显示版本号（如 v18.0.0），说明已安装 ✅
   - 如果显示"command not found"，说明没安装 ❌

**如果没有安装Node.js：**

1. 访问：https://nodejs.org
2. 点击绿色的"LTS"版本下载按钮
3. 下载完成后，双击安装包
4. 按照提示安装（一直点"下一步"）
5. 安装完成后，重启终端
6. 再次输入 `node --version` 检查

---

### 第二步：打开项目文件夹

1. **打开终端**（如果还没打开）

2. **进入项目文件夹**
   - 在终端输入：
   ```bash
   cd ~/Desktop/cursor/crypto-vibe-scorer
   ```
   - 按回车
   - 如果成功，终端会显示项目路径

**如果提示"找不到文件夹"：**
- 检查文件夹是否在：桌面 → cursor → crypto-vibe-scorer
- 或者直接拖拽文件夹到终端窗口，会自动显示路径

---

### 第三步：安装依赖（下载工具）

1. **确保你在项目文件夹里**
   - 终端应该显示类似：`~/Desktop/cursor/crypto-vibe-scorer`

2. **输入安装命令**
   ```bash
   npm install
   ```
   - 按回车

3. **等待下载**
   - 会显示很多下载信息
   - 可能需要3-10分钟
   - 看到"added XXX packages"就完成了 ✅

**如果出错怎么办？**
- 如果显示网络错误，可能是网络问题，多试几次
- 如果显示权限错误，可能需要输入密码

---

### 第四步：运行开发服务器（启动程序）

1. **确保依赖已安装完成**
   - 看到"added XXX packages"就OK

2. **输入启动命令**
   ```bash
   npm run dev
   ```
   - 按回车

3. **等待启动**
   - 会显示一些信息
   - 看到类似这样的信息就成功了：
   ```
   ▲ Next.js 14.0.4
   - Local:        http://localhost:3000
   - Ready in 2.3s
   ```

4. **打开浏览器**
   - 浏览器可能会自动打开
   - 如果没有，手动打开浏览器
   - 访问：http://localhost:3000
   - 应该能看到你的程序界面了！🎉

## ❓ 常见问题

### Q1: 终端是什么？在哪里？

**Mac：**
- 按 `Command + 空格键`
- 输入"终端"或"Terminal"
- 回车打开

**Windows：**
- 按 `Win + R`
- 输入 `cmd`
- 回车打开

### Q2: 输入命令后没反应？

- 检查是否按了回车键
- 命令输入后必须按回车才会执行

### Q3: 显示"command not found"？

- 可能是Node.js没安装
- 或者命令拼写错误
- 检查是否安装了Node.js

### Q4: 安装很慢？

- 正常，可能需要几分钟
- 取决于网络速度
- 耐心等待即可

### Q5: 启动后浏览器打不开？

- 手动打开浏览器
- 输入地址：http://localhost:3000
- 或者点击终端里显示的链接

### Q6: 想停止程序？

- 在终端按 `Ctrl + C`（Mac是 `Control + C`）
- 程序会停止运行

📝 GitHub更新网站 - 3步快速版

方法一：网页版（最简单）

步骤1：打开GitHub
- 访问 https://github.com
- 登录账号
- 找到 my-website 仓库，点击进入

步骤2：删除旧文件
- 找到 index.html，点击
- 点击右上角垃圾桶图标 🗑️
- 输入 delete 确认
- 点击 Commit changes

步骤3：上传新文件
- 点击绿色的 "Add file" 按钮
- 选择 "Upload files"
- 打开桌面 cursor/my-website 文件夹
- 拖拽所有文件到GitHub页面
- 底部写：更新网站设计
- 点击 Commit changes
- 等待完成

步骤4：查看网站
- 等待1-2分钟
- 访问你的网站地址
- 刷新页面查看

方法二：GitHub Desktop（推荐）

步骤1：下载安装
- 访问 https://desktop.github.com
- 下载并安装
- 用GitHub账号登录

步骤2：克隆仓库
- File → Clone repository
- 选择 my-website
- 点击 Clone

步骤3：更新文件
- 点击 Show in Finder
- 把桌面 cursor/my-website 里的文件
- 复制到打开的文件夹里（替换旧文件）

步骤4：提交更新
- 回到GitHub Desktop
- 左下角写：更新网站设计
- 点击 Commit to main
- 点击 Push origin
- 等待完成

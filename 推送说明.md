# 推送代码到 GitHub 的步骤

## 方法一：使用 Personal Access Token（推荐）

### 1. 生成 Token
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" -> "Generate new token (classic)"
3. 设置名称：`crypto-vibe-scorer-deploy`
4. 选择权限：勾选 `repo`（完整仓库权限）
5. 点击 "Generate token"
6. **重要**：复制生成的 token（只显示一次，请保存好）

### 2. 推送代码
在终端执行：

```bash
cd /Users/mima0000/Desktop/cursor/crypto-vibe-scorer

# 推送代码（会要求输入用户名和密码）
git push -u origin main

# 用户名：输入你的 GitHub 用户名（pancake911）
# 密码：输入刚才生成的 Personal Access Token（不是你的 GitHub 密码！）
```

---

## 方法二：使用 GitHub Desktop（最简单）

1. 下载 GitHub Desktop：https://desktop.github.com
2. 登录你的 GitHub 账号
3. 点击 "File" -> "Add Local Repository"
4. 选择项目文件夹：`/Users/mima0000/Desktop/cursor/crypto-vibe-scorer`
5. 点击 "Publish repository" 或 "Push origin"
6. 完成！

---

## 方法三：使用 SSH（适合经常使用）

如果你已经配置了 SSH key，可以改用 SSH 方式：

```bash
cd /Users/mima0000/Desktop/cursor/crypto-vibe-scorer

# 改用 SSH 地址
git remote set-url origin git@github.com:pancake911/crypto-vibe-scorer.git

# 推送
git push -u origin main
```

---

## 推送完成后

1. 访问你的 GitHub 仓库：https://github.com/pancake911/crypto-vibe-scorer
2. 确认 `app` 目录和所有文件都在
3. 回到 Vercel，重新部署（或等待自动部署）

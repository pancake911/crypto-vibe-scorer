# 📦 使用说明 - 给拿到代码的朋友

## ✅ 是的，可以在你的电脑上运行！

只要你的电脑安装了Node.js，就可以运行这个程序。

---

## 🚀 快速开始（3步）

### 第1步：检查是否安装了Node.js

**打开终端（Terminal）**：
- Mac: 按 `Command + 空格键`，输入"终端"，回车
- Windows: 按 `Win + R`，输入 `cmd`，回车

**输入命令检查**：
```bash
node --version
```

**如果显示版本号**（如 `v18.0.0`）：
- ✅ 已安装，跳到第2步

**如果显示"command not found"或"不是内部命令"**：
- ❌ 需要先安装Node.js
- 访问：https://nodejs.org
- 下载并安装LTS版本（绿色按钮）
- 安装完成后，重启终端，再次检查

---

### 第2步：安装依赖（下载工具）

**打开终端，进入项目文件夹**：
```bash
cd 项目文件夹路径
```

**例如，如果项目在桌面**：
```bash
cd ~/Desktop/crypto-vibe-scorer
```

**安装依赖**：
```bash
npm install
```

**等待3-10分钟**，看到 `added XXX packages` 就完成了 ✅

---

### 第3步：启动程序

**在终端输入**：
```bash
npm run dev
```

**等待几秒**，看到类似这样的信息：
```
▲ Next.js 14.0.4
- Local:        http://localhost:3000
- Ready in 2.3s
```

**打开浏览器**，访问：`http://localhost:3000`

**应该能看到程序界面了！** 🎉

---

## ❓ 常见问题

### Q1: 显示"command not found"？
- 说明Node.js没安装
- 去 https://nodejs.org 下载安装

### Q2: 安装依赖很慢？
- 正常，可能需要几分钟
- 取决于网络速度

### Q3: 安装依赖出错？
- 可能是网络问题，多试几次
- 或者使用国内镜像：
  ```bash
  npm install --registry=https://registry.npmmirror.com
  ```

### Q4: 启动后浏览器打不开？
- 手动打开浏览器
- 输入地址：`http://localhost:3000`

### Q5: 想停止程序？
- 在终端按 `Ctrl + C`（Mac是 `Control + C`）

---

## 💡 重要提示

1. **第一次运行**需要安装依赖（`npm install`），可能需要几分钟
2. **之后每次使用**，只需要运行 `npm run dev` 即可
3. **关闭终端**，程序会停止运行
4. **下次使用**，重新运行 `npm run dev` 即可
